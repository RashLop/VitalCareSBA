# VitalCareSBA

`VitalCareSBA` es una solución basada en **SBA (Sistema Basado en Arquitecturas)** compuesta por un frontend web y servicios backend independientes.

El objetivo principal es separar responsabilidades entre proyectos, permitiendo que cada servicio tenga su propia arquitectura interna y que el frontend consuma la información mediante comunicación HTTP/JSON.
